About SCT

The Shariff Complexity Theorem — what it is and why it exists.

The Problem

Existing metrics measure code complexity, not language design complexity. SCT fills this gap by rating how hard a language is to learn — separating syntactic alienness (E) from labor distribution (WD).

The Formula
SCT = 0.7(E) + 0.3(|WD|) + BT
E = 100(1 - √(x²+y²) / 141.4)
|WD| = WD (real) or √(a²+b²) (complex, max 100√2)
BT = 0 or +6 (flat, not inside WD)
Ceiling = 118.4 (Lambda Calculus)
